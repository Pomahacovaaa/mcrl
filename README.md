## What is this project?
This project is my list of useful websites that I like to use when I'm coding or when i want to learn or practice something new. It was made for the Hack Club Sleepover 2026.
## Why did I make it?
I made this website, because i usually forget the names of these sites or I just don't remember where/if I saved them (which google account? tiktok?? somewhere in my notes app???).
So when I was brainstorming about which project i was going to code, this came to my mind... Also this way i can share my favourite websites with others.
## How did i make it?
I made this using HTML and CSS in VS Code and deployed it using GitHub Pages. 
## Features
There are 3 categories:
* data science (dataviz project, sql murder mystery, google courses)
* web design (font websites, css diner, lorem ipsum generator)
* other (codedex)

Each website recommendation consists of:
* the name  
* a short note from me  
* a link  
* a screenshot
## Challenges
